# Prism CRM Dashboard

A modern CRM (Customer Relationship Management) Dashboard UI developed as part of **Future Interns – UI/UX Design Task 3 (2026)**.

## Project Overview

Prism CRM is a modern CRM dashboard designed for a digital marketing agency to manage clients, sales pipelines, daily tasks, campaigns, and analytics from a single platform.

## Features

- Dashboard Overview
- Client Management
- Client Profile
- Sales Pipeline
- Task Manager
- Analytics Dashboard
- Campaign Manager

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React

## Workflow

Lead
→ Pipeline
→ Clients
→ Client Profile
→ Tasks
→ Campaigns
→ Analytics
